# Claude Git Bridge

HTTPS proxy enabling Claude.ai to perform git operations through a secure tunnel to your local machine.

## Quick Start

```bash
# Setup
./scripts/setup.sh

# Start server
./scripts/start_server.sh

# In another terminal, expose with ngrok
ngrok http 8443

# Update .env with ngrok URL
```

## Usage

```python
from git_client import GitProxyClient

client = GitProxyClient()  # Reads from env vars
repo = client.clone('https://github.com/user/repo.git')
client.commit(repo, "Update from Claude")
client.push(repo)
```

## Configuration

Copy `.env.example` to `.env` and set:
- `PROXY_SECRET_KEY` - Auth token (auto-generated by setup)
- `GIT_PROXY_URL` - Your ngrok or custom domain URL

## Architecture

```
Claude.ai Skills → HTTPS → Flask Proxy → Git Commands → GitHub
```

## Security

- Token authentication required
- Only git commands allowed
- Workspace isolation
- Full audit logging

## Use Cases

- **Basic**: Clone, commit, push from Claude.ai
- **Advanced**: Claude manages its own project knowledge repository

See `examples/` for detailed workflows.

## Deployment

- **Local**: ngrok (simplest)
- **Custom domain**: nginx + SSL
- **VPS**: Deploy with systemd service
