# Changelog

All notable changes to the SIFT skill will be documented in this file.

## [1.0.0] - 2025-12-31

### Added
- Initial release of SIFT (Stop, Investigate, Find, Trace) methodology skill
- Complete Deep Background fact-checking instructions from Mike Caulfield
- Structured response format with 8 required sections
- Evidence categorization framework (Documentation, Testimony, Statistics, Analysis, Reporting, Common Knowledge)
- Toulmin analysis method for evaluating claims
- Photo provenance verification procedures
- Special commands: "another round," "context report," "sources table," "read the room"
- Guidance for handling contradictions, competing theories, and information environment assessment
